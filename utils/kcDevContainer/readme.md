# kcDevContainer  
  
`kcDevContainer` is standardized Docker development environment container.

Runs as a non-root user in the latest Debian public image.
  
## Setup    
  
```sh
make build
```    
  
## Usage    
  
Run and enter the container's shell:  
```sh
make run
```
